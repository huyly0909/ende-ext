# EnDe

A quick, offline tool to encrypt and decrypt text using Fernet symmetric encryption right from your browser popup.

## Features
- **Fernet Standard:** Full compatibility with Python's cryptography library (Fernet).
- **Key Management:** Easily generate new keys or paste your existing ones. Keys are stored securely in your browser's local storage.
- **Lightning Fast:** Everything happens directly in the extension popup.
- **100% Offline:** Encryption and decryption happen entirely locally on your device. No data is sent to external servers.

## Installation (Developer Mode)
1. Clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select the cloned folder.
